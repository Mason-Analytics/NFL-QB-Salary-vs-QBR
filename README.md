# NFL-QB-Salary-vs-QBR
Analysis of 2025 NFL QB salary cap hits vs ESPN QBR performance ratings
Is paying a QB more worth it in regards to their performance? This project analyzes 2025 cap hit data from spotrac against ESPN QBR rating for all qualifying QBs.
Key finding: Rookie contracts significantly outperform veteran salaries.
